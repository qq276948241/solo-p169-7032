# ⚔ 文字地牢探险

一个用 Node.js + chalk 写的命令行 Roguelike 文字游戏。

## 🚀 运行

```bash
npm install
npm start
```

## 🎮 核心玩法

随机生成 10~14 个房间组成的地牢迷宫，玩家从入口出发，最终找到出口逃离。

### 指令

| 指令 | 说明 |
|---|---|
| `go [方向]` / `g` | 向 north / south / east / west 移动 |
| `attack` / `a` | 攻击房间里的怪物 |
| `use [物品名]` | 使用血瓶 / 用钥匙开锁 / 装备武器 |
| `open` | 打开宝箱 |
| `flee` / `f` | 战斗中尝试逃跑（50% 成功率） |
| `shop` / `s` | 在商店查看商品 |
| `buy [物品名]` / `b` | 在商店购买物品，例：`buy potion` |
| `inventory` / `i` | 查看状态、金币和背包 |
| `look` / `l` | 查看当前房间 |
| `save` | 保存进度到 `save.json` |
| `exit` | 在出口房间离开地牢（胜利） |
| `help` / `h` | 显示帮助 |

### 🗺️ 房间内容

- **怪物**（红）：哥布林 / 骷髅兵 / 蝙蝠 / 巨型蜘蛛 / 暗影刺客
- **宝箱**（黄）：随机物品，可能上锁需要钥匙
- **陷阱**（灰）：首次踩中扣血
- **商店**（黄边框）：一间独立的安全房间，可消费金币

### 💰 金币与商店

- 击败怪物有 **80% 概率**掉落金币（数量随怪物强度递增）
- 地图上随机生成 **1 间商店房间**（不与怪物、陷阱、宝箱、起点、出口重合）
- 商店商品：

| 商品 | 价格 | 效果 |
|---|---|---|
| 血瓶 | 15 金 | 恢复 30 HP |
| 钥匙 | 10 金 | 打开上锁宝箱 |
| 铁剑 | 30 金 | 攻击力 +5 |
| 战斧 | 50 金 | 攻击力 +8 |
| 巨剑 | 90 金 | 攻击力 +12（商店限定） |

### 💀 死亡机制

HP 归零后会显示 ASCII 骷髅头，然后：
- 扣半血复活在地牢入口
- 死亡次数 +1
- 金币和背包保留

## 💾 存档

游戏存档保存在项目根目录的 `save.json`，下次启动可选「读取存档」继续。
